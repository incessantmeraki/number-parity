# number-parity

No. of bits of an integer determines parity of a number. If it is even, parity is 0, else it's 1. 

## Usage

### Installation

```sh
  $ npm install number-parity
```

### Example

```js
  var parity = require('number-parity')
  console.log(parity(34)) // 0
  console.log(parity(42)) // 1
```
